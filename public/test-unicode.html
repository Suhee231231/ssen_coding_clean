유니코드 문자 테스트

아래첨자 테스트:
H₂O (물), CO₂ (이산화탄소), H₂SO₄ (황산), Ca(OH)₂ (수산화칼슘)
위첨자 테스트:
x² + y² = z², E = mc², 2³ = 8, 10⁴ = 10000
그리스 문자 테스트:
α, β, γ, δ, ε, π, μ, σ, θ, φ
수학 기호 테스트:
±, ×, ÷, ≤, ≥, ≠, ≈, ∞, ∫, ∑, ∏
화학 기호 테스트:
H⁺, OH⁻, Na⁺, Cl⁻, Fe²⁺, Fe³⁺
코드 블록 내 유니코드 테스트:
// 화학식 예시
const water = "H₂O";
const carbonDioxide = "CO₂";
const sulfuricAcid = "H₂SO₄";

// 수학 공식 예시
const pythagorean = "a² + b² = c²";
const einstein = "E = mc²";
